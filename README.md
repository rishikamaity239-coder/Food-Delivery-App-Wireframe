# Food Delivery App Wireframe

## Project Overview

This project contains low-fidelity wireframes for a Food Delivery Mobile Application created in Figma.

## Features

* User Authentication
* Food Search
* Food Categories
* Shopping Cart
* Order Summary
* Delivery Address
* Payment Method
* Order Confirmation
